## Installation
Node.js 10 or higher is required.

Supported frameworks:
- Express
- Adonis.js

More coming soon!

```bash
# For Express
npm i @knuckleswtf/scribe-express

# For Adonis.js
npm i @knuckleswtf/scribe-adonis
```
